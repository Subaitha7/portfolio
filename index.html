TypeSurvival

↗ github ↗ live demo

A real-time adaptive typing game with three escalating difficulty modes — Normal, Heat, and Chaos — featuring matrix rain, particle bursts, and CSS shockwaves tied to live WPM thresholds.

▸ Client-side Trie data structure for O(k) autocomplete — zero server round-trips, sub-millisecond latency
▸ Levenshtein edit-distance in Heat Mode to surface near-miss distractors (edit distance 1–2)
▸ Web Worker offloads vocabulary search to keep main thread non-blocking at 60fps
▸ Custom React hook useTypingMetrics with mutable refs for stale-closure-free WPM/accuracy tracking
▸ Upstash Redis sorted sets for O(log n) leaderboard ranking via serverless Next.js API routes
